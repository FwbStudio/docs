# &#128196; Configuration Files

This section documents the files users normally edit.

## Main Files

| File | Purpose |
|---|---|
| `config/sh_config.lua` | Main shared Bridge configuration |
| `unlocked/client.lua` | Client-side custom override logic |
| `unlocked/server.lua` | Server-side custom override logic |

## In This Section

| Page | What It Covers |
|---|---|
| [Shared Config (`sh_config.lua`)](shared-config.md) | Framework and integration settings that users actually change |

## Notes

- most users mainly work in `sh_config.lua`
- unlocked files are for custom overrides, not normal setup
- locked Bridge files should not be your first edit point
