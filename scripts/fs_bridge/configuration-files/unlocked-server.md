# Unlocked Server (`server.lua`)

`unlocked/server.lua` is the server-side write area for manual compatibility code.

## Current State

In the current Bridge build, this file is intentionally left almost empty except for the write-area markers.

That means it is ready for your own server-side manual compatibility snippets from the docs.

## Use This File For

- server-side manual compatibility
- unsupported server integrations
- override snippets from the Bridge docs

## Important Rule

Write your code inside the intended write area and keep the file structure itself intact.

If you update Bridge later, keep a backup of your unlocked files so you can restore your manual compatibility code after the update.
