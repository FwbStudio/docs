# Client

This page documents client-side text UI helpers.

`lua
FWB.TextUi.Show(text, options)
FWB.TextUi.Hide()
exports['fs_bridge']:ShowTextUi(text, options)
exports['fs_bridge']:HideTextUi()
` 

Common option keys include 	ype, icon, iconColor, position, and iconAnimation.
