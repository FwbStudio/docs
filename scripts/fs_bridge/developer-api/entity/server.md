# Server

This page documents server-side entity search helpers.

Use the same FWB.Entity.* names and exported helpers as client, but pass extras.source when you want Bridge to use a player's ped position as the base point.

Common keys: coords, source, maxDistance, maxCount, sortedByDistance, equireClearLos, model, models, and excludeModels.
