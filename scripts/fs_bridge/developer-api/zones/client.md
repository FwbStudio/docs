# Client

This page documents client-side FWB.zones helpers.

`lua
FWB.zones.poly(data)
FWB.zones.box(data)
FWB.zones.sphere(data)
FWB.zones.getAllZones()
FWB.zones.getCurrentZones()
FWB.zones.getNearbyZones()
` 

poly requires at least 3 points, ox uses coords, size, and otation, and sphere uses coords plus adius.
